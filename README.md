# Teslo API

1. Instalar dependencias
```
npm install
```

2. Clonar el archivo __.env.template__ y renombrarlo a __.env__

3. Cambiar las variables de entorno

4. Levantar la base de datos
```
docker-compose up -d
```
5. Levantar 
```
npm run start:dev
```

6. Ejercutar Seed
```
http://localhost:3005/api/seed
```
